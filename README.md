
## Features Implemented
- **Product List View**
  - Displays: Product Name, Category, Producer, Status (Draft/Submitted/Published), Timestamp
  - Supports **search** by name or text
  - **Filters**: Category and Status
  - **Sort**: Alphabetically by Name, or by Date
  - Polished **cards with hover/focus states**

- **Product Detail View**
  - Shows disclosure summary: who declared, date, evidence count
  - Version history (≥ 2 previous versions)
  - Disclaimer: "This page presents producer-declared information; it is not certification or verification"

- **Interaction & UI**
  - Loading/Skeleton states while data loads
  - Empty states when no products match filters
  - Keyboard navigation support
  - Smooth transitions (150–250ms)
  - Clean, institutional, calm design (like financial or healthcare systems)

- **Language & Messaging**
  - Focused on **producer responsibility**
  - No authority words: “Certified / Verified / Approved / Endorsed”
  - Every UI element communicates disclosure clearly
